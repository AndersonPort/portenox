# Portenox AI

AI-powered platform designed to help professionals optimize their profiles for recruiter visibility, ATS systems, and career growth.

---

## 🚀 Overview

Portenox analyzes professional profiles and provides actionable insights to improve:

- Recruiter visibility
- ATS keyword optimization
- Professional branding
- Career positioning
- Technical skill recommendations

The platform is being built as a modern SaaS application using Next.js, TypeScript and AI-driven analysis workflows.

---

## ⚙️ Getting Started

### 1. Clone repository

```bash
git clone https://github.com/AndersonPort/portenox.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

### 4. Open browser

```bash
http://localhost:3000
```
